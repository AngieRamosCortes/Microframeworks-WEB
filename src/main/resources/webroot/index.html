Framework Features

  • Lambda-based Routes: Define REST services using lambda expressions
  • Query Parameter Access: Easy access to URL parameters with req.getValues()
  • Static File Serving: Configurable static file directory
  • Response Types: Support for JSON, HTML, and plain text responses
  • HTTP Methods: GET, POST, PUT, DELETE support
  • Error Handling: Built-in error responses and status codes

Available Endpoints

GET /hello?name=Pedro Greeting service with query parameter
GET /pi Returns mathematical constant PI in JSON
GET /random Generates a random number
GET /calc?a=10&b=5&op=add Calculator service with multiple parameters
GET /health Health check endpoint
GET /about Framework information page
